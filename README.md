# Translator
划词翻译（在线翻译），默认英文翻译成中文，可根据菜单选择不同目标语言。

![菜单选择]("https://github.com/jiameng123/Translator/blob/master/image/menus.png")

## 使用
1.ctrl + shift + t 激活/关闭插件
2.选择一些要翻译的文本
3.查看翻译语言👍
